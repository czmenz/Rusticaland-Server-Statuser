# Privacy Policy - Rusticaland Server Checker

Rusticaland Server Checker does not collect personal data from users. The tool only performs local diagnostics to check whether your connection to Rusticaland servers is blocked (e.g., by a firewall or VPN).  

## What the tool does
- Checks internet connectivity and Rusticaland server availability.
- Detects VPN/Proxy usage locally and via external API to verify IP address.
- Provides hints to resolve connection issues (firewall rules, VPN usage).

## Data Handling
- No personal information, IP addresses, or sensitive data is stored.
- The tool does not send any data to external parties during or after use.
- After finishing, the tool generates a local log file containing only the necessary information to help diagnose and fix issues with the Checker. The log does **not** include IP addresses or any data that could compromise the user.

## Sharing
- Log files may be shared with Rusticaland admins or support for troubleshooting purposes.
